# 🛡️ Lei Geral de Proteção de Dados (Lei nº 13.709/2018)

Dispõe sobre o tratamento de dados pessoais, inclusive nos meios digitais, por pessoa natural ou por pessoa jurídica de direito público ou privado.

## 🗝️ Fundamentos de Conformidade

### 1. Princípios do Tratamento (Art. 6º)
As atividades de tratamento de dados pessoais devem observar a boa-fé e os seguintes princípios:
- **Finalidade**: Realização do tratamento para propósitos legítimos e específicos.
- **Necessidade**: Limitação do tratamento ao mínimo necessário para a realização de suas finalidades.
- **Transparência**: Garantia aos titulares de informações claras sobre a realização do tratamento.

### 2. Bases Legais (Art. 7º)
O tratamento de dados pessoais somente poderá ser realizado nas seguintes hipóteses:
- Mediante o fornecimento de **consentimento** pelo titular.
- Para o cumprimento de obrigação legal ou regulatória.
- Quando necessário para a execução de contrato.
- Para o exercício regular de direitos.

### 3. Direitos do Titular (Art. 18)
O titular dos dados pessoais tem direito a obter do controlador, a qualquer momento e mediante requisição:
- Confirmação da existência de tratamento.
- Acesso aos dados.
- Correção de dados incompletos ou inexatos.
- **Anonimização**, bloqueio ou eliminação de dados desnecessários.

## 🔍 Implicações para a Themis
A Themis deve auditar se projetos de IA implementam **Privacy-by-Design**:
- O dataset de treinamento utiliza dados anonimizados?
- Existe uma base legal clara (Art. 7º) para a coleta dos dados?
- O sistema permite a auditoria da finalidade do processamento (Art. 6º).
- Crucial para o Score de **Compliance (C)**.
